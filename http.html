Solicitud HTTP

Cuando el navegador ya conoce la dirección IP del servidor, envía un mensaje estructurado llamado "Solicitud HTTP". Los métodos más comunes que utiliza son GET y POST.

Métodos Principales

Método Descripción Cuándo se usa
GET Solicita la representación de un recurso específico. Para leer o consultar
POST Envía datos al servidor de forma segura para procesarlos. Para formularios y registros
